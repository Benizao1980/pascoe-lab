# Pascoe Lab website — redesign v2

Static GitHub Pages site for `pascoelab.com`.

## What is included

- New Wes-Anderson-influenced scientific visual identity
- Expanded research programme
- Project pages for GETcampy, CCC and HU-RIZON
- Selected publications page with Altmetric badges
- Searchable live DOI-indexed publication list (Crossref + ORCID)
- Stories / behind-the-paper archive
- Supervision and opportunities page
- Interactive collaborator map
- Resources and protocol links
- Redirects from the original `latest-work.html`, `news.html` and `lab-guides.html`

## Publish

Upload **the contents of this folder** to the root of the `Benizao1980/pascoe-lab`
repository and commit to `main`. GitHub Pages will deploy automatically.
